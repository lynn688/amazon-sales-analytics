# Data Dictionary - Amazon Sales Dataset

| Column | Type | Description | Example |
|--------|------|-------------|---------|
| order_id | INT | Unique order identifier | 1, 2, 3 |
| order_date | DATE | Purchase date | 4/13/2022 |
| product_id | INT | Product identifier | 2637, 2300 |
| product_category | VARCHAR | Product category | Books, Fashion, Sports |
| price | DECIMAL | Original price | 128.75, 302.60 |
| discount_percent | INT | Discount % | 10, 20, 30 |
| quantity_sold | INT | Units sold | 4, 5, 2 |
| customer_region | VARCHAR | Geographic region | North America, Asia, Europe |
| payment_method | VARCHAR | Payment type | UPI, Credit Card |
| rating | DECIMAL | Product rating (1-5) | 3.5, 3.7, 4.4 |
| review_count | INT | Number of reviews | 443, 475 |
| discounted_price | DECIMAL | Final price after discount | 115.88, 242.08 |
| total_revenue | DECIMAL | Order total revenue | 463.52, 1210.40 |
